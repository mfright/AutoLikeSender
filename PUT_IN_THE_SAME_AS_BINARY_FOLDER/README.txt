■1.本ツールの目的
このソフトは、Instagramで特定ハッシュタグのついたツイートに
自動で「いいね」していくツールです。

■2.動作環境
Windows10 64bit + .netFramework 4.6.2以降。
【日本語】のInstagramアカウント。

■3.使い方
keywords.txtを開き、「いいね」を付けたいハッシュタグ名を1行1つで記載して保存します。(#は含めない)
AutoFav3.exeを起動し、１分以内にログイン画面でログインします。
後は数時間、このウィンドウを閉じずに放置です。

■4.自動動作のしくみ
keywords.txtで指定されたキーワードのハッシュタグを持つ投稿を、
interval.txtで設定された秒(変更可)おきに読み込んでは、
「いいね」します。
キーワードには、有名な(数十秒に1件くらいの頻度で投稿される)ものを使用してください。

一気にやるとロックされてしまうので、インターバルを数秒開けていますが、
interval.txtで、おおよそのインターバル(秒)を長くしたり短くしたりできます。

また、安全の為デフォルトでは数百回いいねしたら停止しますが、
limitter.txtを書き換えればこの制限を変更できます。

exeファイルと同じフォルダに、 under.txt があれば、起動時にウィンドウを下部へ隠します。
under.txt が無かったり違うファイル名なら、隠しません。
